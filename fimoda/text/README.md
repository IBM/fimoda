# Code for SST-2 sentiment classification data

### Files
- **further_train_sst2.py**: Perform further training runs for BERT model on SST-2 dataset
- **lissa_sst2.py**: Run LiSSA (a gradient-based TDA method) on BERT model and SST-2 dataset
- **sst2_bert.ipynb**: Notebook implementing the remaining computations for the SST-2 dataset
- **training_LM.py**: Training functions and classes

### Data and models
The notebook sst2_bert.ipynb contains code for downloading and processing the SST-2 dataset and for downloading and fine-tuning a BERT model.

### Left-out training instances
- **results/sst2/ind_leave_out.pt**: Indices of left-out training instances for the SST-2 dataset
